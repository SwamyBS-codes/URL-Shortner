# URL Shortener Frontend

This client is a polished React + Vite frontend for a URL shortener product.

## Scripts

- `npm run dev` starts the Vite dev server.
- `npm run build` creates a production bundle.
- `npm run lint` runs ESLint.
- `npm run preview` serves the built client locally.

## Notes

- The frontend is styled as a complete landing/dashboard experience.
- The dev server proxies `/api` requests to `http://localhost:3001`.
- Backend boilerplate lives in the root-level `backend` folder.
